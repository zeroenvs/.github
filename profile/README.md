# ZeroEnvs

**Zero-Install development environments powered by containers.**

---

## What is ZeroEnvs?

ZeroEnvs is an open ecosystem of development environment templates designed to eliminate setup friction.

Instead of installing SDKs, runtimes, and tools locally, you spin up fully configured environments using containers.

---

## 🧠 Philosophy

We believe development environments should be:

* **Zero-Install** → No local dependencies beyond Docker
* **Portable** → Works the same across machines
* **Disposable** → Easy to create and destroy
* **Reproducible** → Same environment every time

---

## 🧱 How it works

ZeroEnvs is built around two core components:

### Template Registry

A central repository that lists available environments and defines standards.

### Environment Templates

Each template is an independent repository that defines:

* Docker-based infrastructure
* Setup scripts
* Development workspace

---

## 📦 Getting Started

1. Browse available templates
2. Choose the environment you need
3. Install and run locally
